# Test Nəticələri

## Pozitiv yoxlamalar

1. Şəxsi əşyalar kateqoriyasına daxil oldum.  
   **Nəticə:** Məhsullar siyahısı uğurla göstərildi.

2. Vəziyyəti hissəsində "Yeni" seçdim.  
   **Nəticə:** Yeni məhsullar siyahısı uğurla göstərildi.

3. Yerləşmə hissəsinə daxil olub Ağdam seçdim.  
   **Nəticə:** Yalnız Ağdam rayonu üzrə məhsullar göstərildi.

4. Qiymət hissəsində min 30 AZN, max 50 AZN seçdim.  
   **Nəticə:** Yalnız bu qiymət aralığındakı məhsullar göstərildi.

5. Vəziyyət: Yeni, yerləşmə: Bakı, qiymət: min 5 – max 10 daxil edib axtarış etdim və filtrləri təmizlə düyməsini kliklədim.  
   **Nəticə:** Bütün filtrlər uğurla təmizləndi.

6. Məhsullardan birini seçdim.  
   **Nəticə:** Məhsul səhifəsi uğurla açıldı, məlumatlar ətraflı göstərildi.

7. “Zəng edin” düyməsinə klik etdim.  
   **Nəticə:** Telefon nömrəsi düzgün şəkildə göründü.

8. Səhifənin aşağı hissəsinə endim və yuxarı ox işarəsinə klik etdim.  
   **Nəticə:** Səhifə yuxarı qalxdı.

9. Məhsul şəklinə klik etdim.  
   **Nəticə:** Şəkil böyüdülmüş formada açıldı.

10. Məhsulu sevimlilərə əlavə etdim.  
    **Nəticə:** Login səhifəsi açıldı.

11. Login səhifəsində bütün xanaları boş saxladım.  
    **Nəticə:** “Şifrə boş olmamalıdır” mesajı göstərildi, giriş icra olunmadı.

## Neqativ yoxlamalar

1. Filtr hissəsində “ən yenilər” seçildi.  
   **Nəticə:** Tarix ardıcıllığı düzgün göstərilmədi.

2. Qiymət: min 2000 AZN, max 1000 AZN seçildi.  
   **Nəticə:** Heç bir nəticə tapılmadı.

3. Qiymət: min 5 USD, max 50 USD seçildi və əvvəlcə bahalı kateqoriya ilə test edildi.  
   **Nəticə:** Siyahıda səhv ardıcıllıq müşahidə olundu.

4. Məhsul səhifəsinə daxil olduqda geri qayıtma (back) düyməsi yox idi.  
   **Nəticə:** Yalnız manual keçid ilə geri qayıtmaq mümkündür.

5. VIP işarəsinə klik etdim.  
   **Nəticə:** Texniki xəta mesajı göstərildi.

6. Axtarışa "!!!" simvolu daxil etdim.  
   **Nəticə:** Məhsullar göstərildi, sistem düzgün filtr etmədi.
